📐 七维评分模型说明

综合评分范围 -15 ~ +15，每个维度有独立权重。最终综合评分对应信号等级。

维度核心逻辑权重关键信号（1h标准）
S1 趋势信号MA5 vs MA10均线排列 + 7日涨跌幅±2多头排列+涨>30% = +2；空头排列+跌>15% = -2
S2 OI方向窗口期持仓量变化方向与幅度±2OI涨>20% = +2（新资金入场）；跌>20% = -2（平仓出逃）
S3 价格-OI共振 ⭐价格与OI方向是否一致±3价涨+OI涨 = +3（真实做多）；价涨+OI跌 = -1（空头回补假涨）；价跌+OI涨 = -3（真实做空）
S4 资金费率 ⭐费率绝对值与方向±3负费率 <-0.05% = +3（空头绞杀）；极高>0.07% = -1.5（多头过热）
S5 聪明钱方向大户持仓量多空比（实际仓位）±2大额持仓做多 >2.5 = +2；净空 <0.5 = -2
S6 拥挤度风险大户账户数多空比（逆向指标）±2极度拥挤多头 >4 = -2（反转信号）；极度拥挤空头 <0.5 = +2
S7 量能配合成交量与价格方向是否匹配±1量增价涨 = +1；量减价涨 = -0.5（涨势不可信）

特殊形态识别

⚡ 空头绞杀：负资金费率 + 价格上涨 + OI增加 → 空头被困，挤压做多机会
🔔 高位派发：大幅上涨后OI骤降+费率极高 → 多头出逃，反转风险
🪤 多头陷阱：价格上涨+OI下降+大户净空仓 → 假突破警告
🧠 聪明钱反向：散户偏多但大户实际仓位净空 → 跟随聪明钱方向
💥 OI爆炸：窗口期OI暴增 >80% → 大量新资金涌入，关注方向
